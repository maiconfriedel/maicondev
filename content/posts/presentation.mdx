Olá, tudo bem?

Pra quem não me conhece, meu nome é Maicon Gabriel Friedel, atualmente tenho 22 anos, estou terminando a graduação em Sistemas da Informação, apaixonado por desenvolvimento, rock 'n roll, Magic: The Gathering e video games.

Minhas principais qualificações técnicas são:

- C# com .Net Core
- SQL Server
- NodeJS (aprendendo)
- React e React Native (aprendendo)

Estou em constante aprendizado, sempre tentando aprender coisas novas e me manter atualizado nesse vasto mundo de tecnologias surgindo o tempo todo.

Vou iniciar esse blog, pois percebi que muitos desenvolvedores optam por criar um blog para compartilhar seu conhecimento, e até mesmo aprender, pois que forma melhor de aprender do que escrevendo não é mesmo?!

Esse blog está sendo desenvolvido utilizando o `gatsby-js`, com o plugin `gatsby-theme-blog`.

Para quem não conhece o [GatsbyJS](https://www.gatsbyjs.org/), ele é basicamente um criador de sites estáticos utilizando React, com suporte a vários plugins pré-configurados que facilitam na criação do seu site.
O `gatsby-theme-blog` que estou utlizando é um exemplo de um plugin pré-configurado para um blog pessoal.

Eu ainda estou aprendendo a utilizar ele, então esperem mudanças no site hehehe :D
